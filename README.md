### Setup project

#### Requirements

- php 5.4+
- nodejs
- npm

#### Install packages for Ember.js

    sudo npm install -g ember-precompile
    sudo npm install -g bower

If npm is not installed (Debian/Ubuntu)

    sudo apt-get install nodejs

#### Install PHP-vendors

    composer install
